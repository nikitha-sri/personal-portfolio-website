Nikitha Sri Portfolio Website

A modern and responsive personal portfolio website built using HTML, CSS, and JavaScript to showcase projects, technical skills, certifications, and achievements.

Designed with a clean UI, smooth animations, responsive layouts, and recruiter-focused presentation to create a strong professional impression.
